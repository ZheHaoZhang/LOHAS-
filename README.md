# LOHAS 樂活五線譜

一個使用 **Vue 3 + Vite** 製作的簡單網站，可輸入股票代號並切換顯示 **樂活五線譜** 與 **樂活通道**；後端使用 **FastAPI + yfinance** 取得歷史資料，並提供兩種分析視角：

- **樂活五線譜**：以 **OLS 線性回歸趨勢線 + 殘差標準差** 計算五條平行直線，適合觀察均值回歸區間。
- **樂活通道**：目前先對齊 **xlsm VBA 版本（Preview）**，以 **最近 100 個交易日收盤價的平均與 ±2 倍母體標準差** 建立通道。

## 功能

- 輸入 yfinance 股票代號，例如 `SPY`、`2330`、`00757`、`006208`（台股純數字代號會自動補成 `.TW`）、`AAPL`
- 切換分析模式：`樂活五線譜` / `樂活通道`
- 切換顯示區間：`1 年`、`3 年`、`3.5 年`、`5 年`、`10 年`、`全部`
- 顯示收盤價與五條樂活線：極度樂觀線、相對樂觀線、趨勢線、相對悲觀線、極度悲觀線
- 顯示樂活通道三條線：通道上限（UB）、100 日均線（約 20 週）、通道下限（LB）
- 內建啟動腳本，可一鍵安裝依賴與啟動前後端

## 五線譜公式

- 趨勢線：使用 OLS 線性回歸 `Y = a + bX`
- 相對樂觀線：`TL + 1SD`
- 極度樂觀線：`TL + 2SD`
- 相對悲觀線：`TL - 1SD`
- 極度悲觀線：`TL - 2SD`

其中 `SD` 為價格相對於趨勢線殘差的標準差，因此圖上五條線會保持互相平行。

## 樂活通道公式

- 此模式目前標示為 **Preview**，用來對齊 `五線譜+通道V0.2(VBA版本).xlsm`
- 來源資料：日線收盤價
- 中線：最近 `100` 個交易日收盤價平均（約略對應 20 週）
- 波動度：最近 `100` 個交易日收盤價的母體標準差 `STDEVP`
- 通道上限（UB）：`MA100 + 2 * STDEVP(100日收盤價)`
- 通道下限（LB）：`MA100 - 2 * STDEVP(100日收盤價)`

樂活五線譜適合先看股價是否進入相對高估或低估區；樂活通道則用來確認價格是否仍在常態趨勢內，避免在異常強勢或異常弱勢時過早判斷。前端切換分析模式時，會直接使用同一次 API 回傳中已準備好的兩組資料，不會再次發送請求。

## 快速開始

```bash
./scripts/setup.sh
./scripts/dev.sh
```

## 啟動方式

### 1. 安裝依賴

```bash
./scripts/setup.sh
```

### 2. 一鍵啟動前後端

```bash
./scripts/dev.sh
```

### 3. 分開啟動後端與前端

```bash
./scripts/start-backend.sh
```

後端預設啟動在 `http://127.0.0.1:8000`。

```bash
./scripts/start-frontend.sh
```

前端預設啟動在 `http://127.0.0.1:5173`，已透過 Vite proxy 將 `/api` 轉送到本機後端。

如果預設埠號被佔用：

```bash
API_PORT=8100 FRONTEND_PORT=5175 ./scripts/dev.sh
```

`./scripts/dev.sh` 也會自動往上找可用埠號。

如果後端不是跑在預設位置，也可以在 `frontend` 目錄建立 `.env.local`：

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000
```

## API

### `GET /api/lohas`

查詢參數：

- `symbol`: yfinance 股票代號；台股輸入 `2330`、`00757`、`006208` 這類純數字代號時會自動補成 `.TW`
- `range`: `1y`、`3y`、`3.5y`、`5y`、`10y`、`max`

範例：

```bash
curl "http://127.0.0.1:8000/api/lohas?symbol=SPY&range=3.5y"
```

回傳內容會同時包含：

- `analyses["five-lines"]`
- `analyses["channel"]`
